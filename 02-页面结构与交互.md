# 02-页面结构与交互

## 1. 设计风格

方向：Linear / Vercel 的克制科技感 + Apple 的留白和质感。

设计原则：

- 浅色为主，后续可扩展暗色模式
- 大面积留白
- 轻卡片、轻阴影、低饱和度
- 图片结果区视觉权重最高
- 表单参数尽量安静，不抢图片注意力
- 主色用黑 / 灰 / 少量品牌强调色
- 不做花哨渐变和复杂动效

## 2. 一级页面

### 2.1 生成工作台

核心页面。

结构建议：

- 顶部：系统名称、模式 Tab、管理入口
- 左侧：参数面板
- 中间：结果展示区
- 右侧或底部：任务队列
- 下方：最近历史记录快捷区

模式 Tab：

1. 文生图
2. 图生图

### 2.2 历史记录

默认时间流。

顶部筛选：

- 关键词搜索
- 模式筛选
- 模板筛选
- 时间筛选（可选）

卡片操作：

- 下载
- 复制 prompt
- 再生成
- 用这张图生成

### 2.3 模板管理

功能：

- 查看用途模板、平台模板、公司自定义模板
- 新增公司模板
- 编辑模板名称、分类、默认 prompt、默认尺寸、默认高级参数
- 从历史记录保存为模板

### 2.4 管理统计

仅管理视图。

模块：

- 今日生成次数
- 本周生成次数
- 成功 / 失败次数
- 预估成本
- 热门模板排行

## 3. 核心流程

### 3.1 文生图流程

1. 进入工作台，选择“文生图”
2. 选择模板或直接输入 prompt
3. 选择尺寸、数量
4. 可展开高级参数
5. 点击生成
6. 任务进入队列
7. 完成后在结果区宫格展示
8. 可下载、再生成、复制 prompt、保存模板

### 3.2 图生图流程

1. 进入“图生图”
2. 上传参考图
3. 输入 prompt
4. 设置参考强度 / 相似度
5. 点击生成
6. 完成后展示原图 vs 新图对比

### 3.3 会话内继续生成流程

1. 从工作台上传图片，或从历史记录点击“用这张图生成”
2. 可开启“会话固定提示词”，把一套长期处理规则固定在当前会话
3. 上传本次主图，也可以追加参考图
4. 输入本次补充说明；如果已开启固定提示词，本次说明可为空
5. 设置参考强度 / 风格强度
6. 按图生图生成并对比结果

### 3.4 会话固定提示词批处理流程

1. 在会话窗口设置固定提示词，例如统一白底电商主图、统一品牌海报风格
2. 后续每次上传图片时，第一张默认作为主图，其他图片可切换为参考图
3. 系统按“固定提示词 + 主图 + 参考图 + 本次补充说明”创建任务
4. 每条消息展示本次使用的固定提示词快照，方便回溯
5. 固定提示词可保存为模板，用于新会话复用

## 4. 任务队列体验

状态：

- 排队中
- 生成中
- 完成
- 失败

展示字段：

- 缩略图 / 模式
- prompt 摘要
- 状态
- 创建时间
- 操作：查看结果 / 取消（可后续做）
