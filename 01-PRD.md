# 01-PRD：公司画境工坊

## 1. 背景

公司需要一个画境工坊，统一调用 sub2api 的 image-2 模型能力，降低员工使用 AI 生成图片的门槛，并沉淀可复用的图片、提示词和模板。

## 2. 产品定位

一个面向全公司的通用图片生成工作台。

核心目标不是做复杂设计软件，而是让员工快速完成：

- 文生图
- 图生图
- 基于图片再创作
- 历史图片复用
- 常用模板复用
- 管理侧查看用量和成本

## 3. 目标用户

- 电商运营：商品图、场景图、详情图素材
- 内容团队：小红书、公众号、短视频封面
- 投放团队：广告图、活动海报
- 其他员工：通用创意图片生成

MVP 按“全公司通用工具型”设计，不为单一部门深度定制。

## 4. MVP 功能范围

### 4.1 生成工作台

两个 Tab：

1. 文生图
2. 图生图

常用参数默认展示：

- prompt
- 尺寸
- 生成数量：1 / 2 / 4，默认 1
- 模板选择
- 参考图上传（图生图）

高级参数默认折叠：

- 负面提示词
- 参考强度 / 相似度
- 风格强度

### 4.2 模板体系

模板类型：

- 用途模板：商品图、海报、封面、头像、背景图等
- 平台模板：小红书封面、公众号头图、抖音封面、朋友圈海报等
- 公司自定义模板：支持管理页手动创建，也支持从历史记录保存为模板

### 4.3 任务队列

工作台内常驻任务队列，不单独做一级页面。

支持：

- 同时提交多个任务
- 展示状态：排队中 / 生成中 / 完成 / 失败
- 完成后进入历史记录

### 4.4 结果展示

- 文生图：宫格展示，方便挑图
- 图生图：原图 vs 新图对比；会话内继续修改仍归类为图生图

### 4.5 历史记录

默认按生成时间倒序。

支持：

- 按模式筛选：文生图 / 图生图
- 按模板筛选
- 按关键词搜索 prompt
- 复制 prompt
- 再生成
- 用这张图生成
- 下载图片

### 4.6 管理统计

仅管理视图可见。

指标：

- 今日 / 本周生成次数
- 成功次数 / 失败次数
- 预估成本
- 热门模板

## 5. 明确不做

MVP 不做以下能力：

- 用户登录
- 访问码 / 内网限制 / URL 密钥
- 详细失败兜底展示
- 素材合规、隐私、存储过期、一键清空历史
- 复杂项目 / 文件夹式图库管理
- seed 等专业参数
